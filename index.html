Bibliotecas

O que são bibliotecas ?

Bibliotecas são um conjunto de funções pré-escritas por outros programadores que já resolvem determinado problema para que outros programadores possam resolver esses problemas sem precisar inventar um código. Existem várias biblitecas dentro de uma linguagem de programação javaScript e vamos focar nesse documento 2 bibliotecas muito comuns na programação de um software. Sendo elas:

  • Math
  • String

Biblioteca Math

O Math é um objeto global embutido que tem propriedades e métodos para constantes e funções matemáticas. Todas as propriedades dos métodos de Math são estáticos. Você pode tanto referenciar a constante PI usando Math e você pode chamar a função para calcular a potência Math.pow(x, y), onde x e y são argumentos da função.

Propriedades

  • Math.E

    • Constante de Euler e base dos logaritmos naturais, aproximadamente 2.718.

  • Math.Ln2

    • Logaritmo natural de 2, aproximadamente 0.693.

  • Math.PI

    • Relação entre a circunferência de um círculo e o seu diâmetro, aproximadamente 3.14159.

Métodos

  • Math.abs(x)

    • Retorna o módulo, ou valor absoluto, de um número (|x|).

  • Math.cbrt(x)

    • Retorna a raiz cúbica de um número.

  • Math.floor(x)

    • Retorna o maior inteiro que é menor ou igual a um número.

  • Math.random()

    • Retorna um valor aletório entre 0 e 1 mas sempre excluindo o 1.

    Para aumentar o valor máximo do sorteio basta multiplicar pelo número que deseja usar como limite, sempre se atentando que esse número será excluido toda vez que randomizar um número.

    Os valores que esse método retorna são valores do tipo float(possuem ponto flutuante), para transformar esses valores quebrados em inteiros pode se usar o método parseInt(), ou utilizar métodos da biblioteca Math de arredondamento como já citado antes neste documento o Math.floor().

  • Math.trunc(x)

    • Esse método retorna um valor sem quaisquer números fracionários, retorna só a parte inteira.

Exercicios:

  1. faça um programa insira 2 números e sorteie um valor entre eles.
  2. faça um programa que você insira as medidas dos lados de um triângulo e calcule o seno, cosseno e tangente deste triângulo. Obs.: as medidas necessitam estar em radianos.
  3. faça um programa em que você insira o diâmetro de um círculo e calcule a área. Em seguida insira a altura e calcule o volume do mesmo círculo.
  4. faça um programa que insira as coordenadas de dois pontos em um espaço 10x10 e calcule a distância Manhattan.

Biblioteca String

String é um objeto global para a contrução de Strings, ou sequência de caracteres. Strings são úteis para guardar dados que são representados em forma de texto. A representação de uma String é feita da seguinte forma:

Acessando caracteres

Uma das maneiras de acessar um determinado caractere de uma string é utilizando o método charAt

Concatenando Strings

O ato de concatenar é quando dois textos são juntos por meio do operador de soma "+"

Métodos

  • String.prototype.toLowerCase()

    • Este método retorna o valor da String todo em minúsculos.

  • String.prototype.toUpperCase()

    • Este método retorna o valor da String todo em maiúsculos.

  • String.prototype.replace(x, y)

    • Este método utiliza dois parâmetros que são duas Strings, uma que irá ser localizada e a outra que irá substituir a que foi localizada gerando uma nova string.
